ryanism.org legacy quotes
==================================

---------- Explanation -----------

This the collection of the old quotes for the website that have become inconceivable.

----------- How to Edit ----------

Only edit the index.json file since the min/index.json is auto generated from compress-json.sh shell script with GitHub actions.

Use escape characters for the quotation marks so it renders properly on each project. DO NOT use Markdown styling in this directory.

------------- Dependents -----------
- None at the moment
